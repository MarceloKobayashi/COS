## 🎯 Visão Geral

O COS é uma aplicação em Oracle APEX criada para controlar e centralizar as Ordens de Serviço (OS) do setor ASQUALOG. Fornece visualização, cadastro e relatórios das OS registradas.

## 🏛️ Contexto e Origem

O projeto nasceu como iniciativa interna do Senado Federal para permitir desenvolvimento descentralizado usando Oracle APEX. A ideia inicial foi proposta pela estagiária Iasmin Soares de Sousa para melhorar o controle das OS do setor ASQUALOG. O desenvolvimento foi iniciado por Marcelo Honda Kobayashi.

## ❗ Problema que resolve

Antes do COS o controle era feito em planilhas Excel que frequentemente ficavam desatualizadas, ocasionando perda de informações e solicitações duplicadas. O COS centraliza o registro e facilita o acompanhamento das solicitações.

## 🎯 Objetivos

- Centralizar o registro das Ordens de Serviço
- Evitar solicitações duplicadas e perda de informação
- Permitir filtros e relatórios por período e responsável
- Facilitar cadastro, edição e remoção de OS

## 🧩 Funcionalidades principais

- Listagem de todas as Ordens de Serviço com filtros
- Cadastro de novas OS
- Edição e exclusão de OS existentes
- Relatórios mensais e por responsável
- Filtros avançados para visualização personalizada
